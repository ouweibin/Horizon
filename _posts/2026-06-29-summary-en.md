## [Historical Memory Prices 1960-2026 Visualized](https://dam.stanford.edu/memory-prices.html) ⭐️ 8.0/10

Stanford University has published a comprehensive dataset and visualization tracking memory prices from 1960 to 2026, covering magnetic core memory, DRAM, and other technologies. The dataset is an updated version of the defunct jcmit dataset, now hosted on dam.stanford.edu. This resource provides a long-term historical perspective on memory pricing, enabling analysis of cost trends, inflation effects, and the impact of technological shifts. It is valuable for researchers, historians, and hardware enthusiasts seeking to understand the economics of memory. The dataset includes prices per bit, per byte, and per GB, but earlier data (before 1990) for GB-equivalent pricing is considered unrealistic since GB-scale systems did not exist then. The visualization does not adjust for inflation, which significantly understates prices in the 1960s-1980s.

**Background**: Memory technology has evolved from magnetic core memory (dominant 1955-1975) to semiconductor DRAM (from the 1970s onward). Core memory was non-volatile but expensive: around $1 per bit in the 1960s, dropping to 1 cent per bit by the late 1960s. DRAM later drove prices down dramatically, though price spikes occur due to demand cycles (e.g., AI-driven demand in 2026). Understanding these trends helps contextualize modern memory costs and consumption.
